# Greek Keyboard

Many keys correspond to upper case glyphs:

- A → Α/α
- B → Β/β
- E → Ε/ε
- H → Η/η
- I → Ι/ι
- K → Κ/κ
- M → Μ/μ
- N → Ν/ν
- O → Ο/ο
- T → Τ/τ
- X → Χ/χ
- Y → Υ/υ
- Z → Ζ/ζ

Others correspond to the _sound_ of the letters:

- D → Δ/δ
- F → Φ/φ
- G → Γ/γ
- L → Λ/λ
- P → Π/π
- R → Ρ/ρ
- S → Σ/σ

Just four keys really need to be memorized:

- C → Ψ/ψ
- J → Ξ/ξ
- U → Θ/θ
- V → Ω/ω

To get a word-ending ς an unshifted W is pressed:

W → ς

Q → :/;
W → ΅/ς
: → ¨
; → ΄
